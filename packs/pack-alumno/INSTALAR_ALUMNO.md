# Pack instalador - Alumno

## URL de instalacion

Abrir desde el celular:

```text
https://bjcbaigo.github.io/qrasisispi4019/app-alumno/
```

## Android

1. Abrir la URL con Chrome.
2. Tocar el menu de tres puntos.
3. Elegir `Agregar a pantalla principal`.
4. Confirmar.
5. Abrir el icono `Alumno ISPI`.

## iPhone

1. Abrir la URL con Safari.
2. Tocar el boton compartir.
3. Elegir `Agregar a pantalla de inicio`.
4. Confirmar.
5. Abrir el icono `Alumno ISPI`.

## Primer uso

1. Ingresar el token institucional entregado por la institucion.
2. Ingresar nombre visible.
3. Presionar `GUARDAR Y MOSTRAR QR`.
4. Mostrar el QR al tutor al llegar al lugar de pasantia.

## Importante

El token reemplaza al DNI dentro del QR. No compartir capturas del QR ni el token.
