# Pack instalador - Tutor

## URL de instalacion

Abrir desde el celular:

```text
https://bjcbaigo.github.io/qrasisispi4019/app-responsable/
```

## Android

1. Abrir la URL con Chrome.
2. Tocar el menu de tres puntos.
3. Elegir `Agregar a pantalla principal`.
4. Confirmar.
5. Abrir el icono `Tutor ISPI`.

## iPhone

1. Abrir la URL con Safari.
2. Tocar el boton compartir.
3. Elegir `Agregar a pantalla de inicio`.
4. Confirmar.
5. Abrir el icono `Tutor ISPI`.

## Primer uso

1. Ingresar la URL del servidor Apps Script.
2. Ingresar el token del tutor.
3. Presionar `Guardar`.
4. Presionar `Escanear QR`.
5. Escanear el QR del alumno.

## Datos que debe entregar la institucion

```text
URL servidor Apps Script:
Token del tutor:
Lugar asignado:
```

No compartir el token del tutor.
